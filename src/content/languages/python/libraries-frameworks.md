Python has a vast ecosystem spanning web, data science, machine learning, and automation.

**Web Frameworks** - [Django](https://www.djangoproject.com/), [Flask](https://flask.palletsprojects.com/), [FastAPI](https://fastapi.tiangolo.com/), [Starlette](https://www.starlette.io/), [Litestar](https://litestar.dev/)

**Data Science** - [NumPy](https://numpy.org/), [Pandas](https://pandas.pydata.org/), [Polars](https://pola.rs/), [SciPy](https://scipy.org/), [Matplotlib](https://matplotlib.org/), [Seaborn](https://seaborn.pydata.org/), [Plotly](https://plotly.com/python/)

**Machine Learning** - [scikit-learn](https://scikit-learn.org/), [PyTorch](https://pytorch.org/), [TensorFlow](https://www.tensorflow.org/), [Keras](https://keras.io/), [Hugging Face Transformers](https://huggingface.co/docs/transformers/index), [JAX](https://jax.readthedocs.io/)

**LLM / AI** - [LangChain](https://www.langchain.com/), [LlamaIndex](https://www.llamaindex.ai/), [OpenAI SDK](https://github.com/openai/openai-python), [Anthropic SDK](https://github.com/anthropics/anthropic-sdk-python)

**HTTP / API** - [Requests](https://requests.readthedocs.io/), [httpx](https://www.python-httpx.org/), [aiohttp](https://docs.aiohttp.org/), [urllib3](https://urllib3.readthedocs.io/)

**Databases / ORM** - [SQLAlchemy](https://www.sqlalchemy.org/), [Django ORM](https://docs.djangoproject.com/en/stable/topics/db/), [Tortoise ORM](https://tortoise.github.io/), [Peewee](https://docs.peewee-orm.com/), [databases](https://www.encode.io/databases/)

**CLI** - [Click](https://click.palletsprojects.com/), [Typer](https://typer.tiangolo.com/), [argparse](https://docs.python.org/3/library/argparse.html), [Rich](https://rich.readthedocs.io/)

**Async** - [asyncio](https://docs.python.org/3/library/asyncio.html), [trio](https://trio.readthedocs.io/), [anyio](https://anyio.readthedocs.io/), [uvloop](https://github.com/MagicStack/uvloop)

**Testing** - [pytest](https://docs.pytest.org/), [unittest](https://docs.python.org/3/library/unittest.html), [hypothesis](https://hypothesis.readthedocs.io/), [coverage](https://coverage.readthedocs.io/)

**DevOps / Automation** - [Fabric](https://www.fabfile.org/), [Ansible](https://www.ansible.com/), [Boto3 (AWS)](https://boto3.amazonaws.com/v1/documentation/api/latest/index.html), [Celery](https://docs.celeryq.dev/)

**Type Checking** - [mypy](https://mypy.readthedocs.io/), [pyright](https://github.com/microsoft/pyright), [Pydantic](https://docs.pydantic.dev/)
