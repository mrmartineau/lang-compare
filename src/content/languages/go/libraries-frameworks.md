Go has a rich standard library and a growing ecosystem focused on cloud, networking, and backend services.

**Web Frameworks** - [Gin](https://gin-gonic.com/), [Echo](https://echo.labstack.com/), [Fiber](https://gofiber.io/), [Chi](https://go-chi.io/), [net/http (stdlib)](https://pkg.go.dev/net/http)

**API / RPC** - [gRPC](https://grpc.io/), [Connect](https://connectrpc.com/), [Twirp](https://twitchtv.github.io/twirp/docs.html), [OpenAPI generators](https://openapi-generator.tech/)

**Databases** - [GORM](https://gorm.io/), [sqlx](https://github.com/jmoiron/sqlx), [pgx](https://github.com/jackc/pgx), [ent](https://entgo.io/), [database/sql (stdlib)](https://pkg.go.dev/database/sql)

**Cloud / Infra** - [Docker](https://www.docker.com/), [Kubernetes client-go](https://github.com/kubernetes/client-go), [Terraform Plugin Framework](https://developer.hashicorp.com/terraform/plugin/framework), [AWS SDK](https://aws.github.io/aws-sdk-go-v2/)

**CLI** - [Cobra](https://cobra.dev/), [urfave/cli](https://github.com/urfave/cli), [Bubble Tea (TUI)](https://github.com/charmbracelet/bubbletea), [Charm tools](https://charm.sh/)

**Logging** - [slog (stdlib)](https://pkg.go.dev/log/slog), [Zap](https://github.com/uber-go/zap), [zerolog](https://github.com/rs/zerolog), [logrus](https://github.com/sirupsen/logrus)

**Config** - [Viper](https://github.com/spf13/viper), [envconfig](https://github.com/kelseyhightower/envconfig), [koanf](https://github.com/knadh/koanf)

**Testing** - [testify](https://github.com/stretchr/testify), [gomock](https://github.com/uber-go/mock), [ginkgo](https://onsi.github.io/ginkgo/), [go-cmp](https://github.com/google/go-cmp)

**Concurrency** - [errgroup](https://pkg.go.dev/golang.org/x/sync/errgroup), [semaphore (x/sync)](https://pkg.go.dev/golang.org/x/sync/semaphore), [conc](https://github.com/sourcegraph/conc)

**Serialization** - [encoding/json (stdlib)](https://pkg.go.dev/encoding/json), [json-iterator](https://github.com/json-iterator/go), [protobuf](https://protobuf.dev/)

**HTTP Clients** - [net/http (stdlib)](https://pkg.go.dev/net/http), [resty](https://github.com/go-resty/resty), [req](https://github.com/imroc/req)
