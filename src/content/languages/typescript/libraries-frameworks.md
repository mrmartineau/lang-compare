TypeScript uses the same ecosystem as JavaScript. These libraries are TypeScript-first or have excellent type support:

**Type-First Libraries** - [Zod (schema validation)](https://zod.dev/), [tRPC (end-to-end typesafe APIs)](https://trpc.io/), [Drizzle ORM](https://orm.drizzle.team/), [Effect](https://effect.website/), [ts-pattern](https://github.com/gvergnaud/ts-pattern)

**Web Frameworks** - [Next.js](https://nextjs.org/), [Nuxt](https://nuxt.com/), [SvelteKit](https://kit.svelte.dev/), [Remix](https://remix.run/), [Astro](https://astro.build/), [Hono](https://hono.dev/) - all have first-class TS support

**Full-Stack** - [T3 Stack (Next.js + tRPC + Prisma + Tailwind)](https://create.t3.gg/), [Blitz.js](https://blitzjs.com/), [RedwoodJS](https://redwoodjs.com/)

**Runtime/Build** - [tsx](https://tsx.is/), [tsup](https://tsup.egoist.dev/), [unbuild](https://github.com/unjs/unbuild), [Vite](https://vite.dev/), [esbuild](https://esbuild.github.io/)

**State Management** - [Zustand](https://zustand-demo.pmnd.rs/), [Jotai](https://jotai.org/), [XState](https://stately.ai/docs/xstate) - all TypeScript-first

**Validation** - [Zod](https://zod.dev/), [Valibot](https://valibot.dev/), [ArkType](https://arktype.io/), [io-ts](https://github.com/gcanti/io-ts), [Yup](https://github.com/jquense/yup)

**API Clients** - [Axios](https://axios-http.com/), [ky](https://github.com/sindresorhus/ky), [openapi-typescript](https://openapi-ts.dev/), [GraphQL Code Generator](https://the-guild.dev/graphql/codegen)

**Type Utilities** - [type-fest](https://github.com/sindresorhus/type-fest), [ts-essentials](https://github.com/ts-essentials/ts-essentials), [utility-types](https://github.com/piotrwitek/utility-types)

**Testing** - [Vitest](https://vitest.dev/), [Jest](https://jestjs.io/), [Playwright](https://playwright.dev/) - all TypeScript-native
