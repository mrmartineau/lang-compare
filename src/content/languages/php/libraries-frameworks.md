PHP has a mature ecosystem with powerful frameworks and libraries:

**Web Frameworks** - [Laravel](https://laravel.com/), [Symfony](https://symfony.com/), [Slim](https://www.slimframework.com/), [CodeIgniter](https://codeigniter.com/), [CakePHP](https://cakephp.org/), [Laminas (Zend)](https://getlaminas.org/)

**CMS** - [WordPress](https://wordpress.org/), [Drupal](https://www.drupal.org/), [Joomla](https://www.joomla.org/), [Craft CMS](https://craftcms.com/), [Statamic](https://statamic.com/)

**API** - [API Platform (Symfony)](https://api-platform.com/), [Laravel API Resources](https://laravel.com/docs/eloquent-resources), [Lumen](https://lumen.laravel.com/)

**Database / ORM** - [Eloquent (Laravel)](https://laravel.com/docs/eloquent), [Doctrine](https://www.doctrine-project.org/), [Propel](https://propelorm.org/), [RedBeanPHP](https://redbeanphp.com/)

**Templating** - [Blade (Laravel)](https://laravel.com/docs/blade), [Twig (Symfony)](https://twig.symfony.com/), [Plates](https://platesphp.com/)

**HTTP Clients** - [Guzzle](https://docs.guzzlephp.org/), [Symfony HttpClient](https://symfony.com/doc/current/http_client.html), [cURL wrapper](https://www.php.net/manual/en/book.curl.php)

**Testing** - [PHPUnit](https://phpunit.de/), [Pest](https://pestphp.com/), [Mockery](https://github.com/mockery/mockery), [Codeception](https://codeception.com/)

**Utilities** - [Carbon (dates)](https://carbon.nesbot.com/), [Flysystem (filesystem)](https://flysystem.thephpleague.com/), [Monolog (logging)](https://github.com/Seldaek/monolog), [League packages](https://thephpleague.com/)

**Queue / Async** - [Laravel Queues](https://laravel.com/docs/queues), [Symfony Messenger](https://symfony.com/doc/current/messenger.html), [ReactPHP](https://reactphp.org/), [Swoole](https://www.swoole.co.uk/)

**Admin Panels** - [Filament](https://filamentphp.com/), [Nova](https://nova.laravel.com/), [Backpack](https://backpackforlaravel.com/)
