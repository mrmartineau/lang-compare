Use `npm init` for a basic project or a framework CLI for a full setup.

```bash
# Basic project
mkdir my-project && cd my-project
npm init -y

# Using Vite
npm create vite@latest my-app

# Using Next.js
npx create-next-app@latest my-app

# Using Astro
npm create astro@latest

# Run the project
npm run dev
```
