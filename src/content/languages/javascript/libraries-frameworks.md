JavaScript has an enormous ecosystem. Key libraries and frameworks by category:

**Web Frameworks** - [React](https://react.dev/), [Vue](https://vuejs.org/), [Svelte](https://svelte.dev/), [Angular](https://angular.dev/), [Solid](https://www.solidjs.com/), [Astro](https://astro.build/), [Next.js](https://nextjs.org/), [Nuxt](https://nuxt.com/), [SvelteKit](https://kit.svelte.dev/), [Remix](https://remix.run/)

**Server / API** - [Express](https://expressjs.com/), [Fastify](https://fastify.dev/), [Hono](https://hono.dev/), [Koa](https://koajs.com/), [Nest.js](https://nestjs.com/), [tRPC](https://trpc.io/)

**Build Tools** - [Vite](https://vite.dev/), [esbuild](https://esbuild.github.io/), [Rollup](https://rollupjs.org/), [webpack](https://webpack.js.org/), [Turbopack](https://turbo.build/pack), [Parcel](https://parceljs.org/)

**State Management** - [Zustand](https://zustand-demo.pmnd.rs/), [Jotai](https://jotai.org/), [Redux Toolkit](https://redux-toolkit.js.org/), [Pinia (Vue)](https://pinia.vuejs.org/), [Signals](https://preactjs.com/guide/v10/signals)

**Styling** - [Tailwind CSS](https://tailwindcss.com/), [Styled Components](https://styled-components.com/), [CSS Modules](https://github.com/css-modules/css-modules), [UnoCSS](https://unocss.dev/)

**Databases / ORM** - [Prisma](https://www.prisma.io/), [Drizzle](https://orm.drizzle.team/), [Mongoose](https://mongoosejs.com/), [Knex](https://knexjs.org/), [TypeORM](https://typeorm.io/)

**Utilities** - [Lodash](https://lodash.com/), [date-fns](https://date-fns.org/), [Zod](https://zod.dev/), [Axios](https://axios-http.com/), [Ramda](https://ramdajs.com/)

**Testing** - [Vitest](https://vitest.dev/), [Jest](https://jestjs.io/), [Playwright](https://playwright.dev/), [Cypress](https://www.cypress.io/), [Testing Library](https://testing-library.com/)

**Desktop** - [Electron](https://www.electronjs.org/), [Tauri](https://tauri.app/)

**Mobile** - [React Native](https://reactnative.dev/), [Expo](https://expo.dev/), [Capacitor](https://capacitorjs.com/), [NativeScript](https://nativescript.org/)
