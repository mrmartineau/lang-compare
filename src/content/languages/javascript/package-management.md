npm is the default package manager. Yarn, pnpm, and Bun are popular alternatives.

```bash
# Install a package
npm install lodash

# Install as dev dependency
npm install --save-dev vitest

# Install globally
npm install -g typescript

# Using pnpm
pnpm add lodash

# Using yarn
yarn add lodash

# Using bun
bun add lodash
```
