Ruby has a rich ecosystem centered around web development and developer tooling:

**Web Frameworks** - [Rails](https://rubyonrails.org/), [Sinatra](https://sinatrarb.com/), [Hanami](https://hanamirb.org/), [Roda](https://roda.jeremyevans.net/), [Grape (API)](https://github.com/ruby-grape/grape)

**Background Jobs** - [Sidekiq](https://sidekiq.org/), [GoodJob](https://github.com/bensheldon/good_job), [Solid Queue](https://github.com/rails/solid_queue), [Delayed Job](https://github.com/collectiveidea/delayed_job), [Resque](https://github.com/resque/resque)

**Database / ORM** - [ActiveRecord (Rails)](https://guides.rubyonrails.org/active_record_basics.html), [Sequel](https://sequel.jeremyevans.net/), [ROM (Ruby Object Mapper)](https://rom-rb.org/)

**API** - [GraphQL Ruby](https://graphql-ruby.org/), [Grape](https://github.com/ruby-grape/grape), [JSONAPI::Resources](https://jsonapi-resources.com/), [jbuilder](https://github.com/rails/jbuilder)

**Authentication** - [Devise](https://github.com/heartcombo/devise), [Rodauth](https://rodauth.jeremyevans.net/), [OmniAuth](https://github.com/omniauth/omniauth), [Sorcery](https://github.com/Sorcery/sorcery)

**Testing** - [RSpec](https://rspec.info/), [Minitest](https://github.com/minitest/minitest), [Capybara](https://github.com/teamcapybara/capybara), [FactoryBot](https://github.com/thoughtbot/factory_bot), [VCR](https://github.com/vcr/vcr)

**HTTP Clients** - [Faraday](https://github.com/lostisland/faraday), [HTTParty](https://github.com/jnunemaker/httparty), [Net::HTTP (stdlib)](https://docs.ruby-lang.org/en/master/Net/HTTP.html), [Typhoeus](https://github.com/typhoeus/typhoeus)

**CLI** - [Thor](https://github.com/rails/thor), [TTY toolkit](https://ttytoolkit.org/), [OptionParser (stdlib)](https://docs.ruby-lang.org/en/master/OptionParser.html)

**Utilities** - [Nokogiri (HTML/XML)](https://nokogiri.org/), [Pry (debugging)](https://github.com/pry/pry), [Dry-rb (functional patterns)](https://dry-rb.org/), [ActiveSupport](https://guides.rubyonrails.org/active_support_core_extensions.html)

**DevOps** - [Capistrano (deployment)](https://capistranorb.com/), [Chef](https://www.chef.io/), [Puppet](https://www.puppet.com/), [Vagrant](https://developer.hashicorp.com/vagrant)
